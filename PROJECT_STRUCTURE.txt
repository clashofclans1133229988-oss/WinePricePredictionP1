================================================================================
                        WORKING WITH THIS STRUCTURE
================================================================================

RUNNING LOCALLY:

1. Install dependencies:
   pip install -r requirements.txt

2. Start Flask app:
   python app.py

3. Visit:
   http://localhost:5000


BUILDING DOCKER CONTAINER:

1. Build image:
   docker build -t wine-app:latest .

2. Run container:
   docker run -p 5000:5000 wine-app:latest

3. Visit:
   http://localhost:5000


COMMITTING TO GITHUB:

1. Stage all files:
   git add .

2. Commit:
   git commit -m "Organize project structure"

3. Push:
   git push origin main

4. GitHub Actions automatically:
   - Runs tests (ci.yml)
   - Builds Docker (docker.yml)
   - Deploys to platform (deploy-railway.yml or deploy-fly.yml)
